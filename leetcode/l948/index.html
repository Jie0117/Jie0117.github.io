948. Bag of Tokens

March 9, 2024

—

題目敘述

由於原題目敘述不好理解，這裡用自己易理解的方式陳述。給予一個陣列，陣列包含購買一個積分(score)所需要的(能量)power，目標是求max_score。 我們可以做的事情有兩件

(1)使用1score換取陣列中所標示的power

token[i]=500 before after
score 1 0
power 0 500

(2)使用power換取1score

token[i]=500 before after
score 0 1
power 500 0

想法

  1. 先對陣列進行升序排序，簡化後續操作
  2. 使用雙指針head和tail分別指向最小和最大的代幣值
  3. 當前power足夠時,從head指針處獲取score, score加1
  4. 當前power不足時,如果score大於0,則從tail指針處出售1score, 獲取power
  5. 重複3和4,直到無法獲取或出售score為止
  6. 返回在此過程中獲得的最高分數 最終只需將整個陣列遍歷一次就可得出結果

code

int  bagOfTokensScore(vector<int>&  tokens, int  power) {
	sort(tokens.begin(), tokens.end());
	int score = 0;
	int maxScore = 0;
	int head = 0;
	int tail = tokens.size() - 1;
	while (head <= tail) 
	{
		if (power >= tokens[head]) 
		{
			power -= tokens[head++];
			score++;
			maxScore = max(maxScore, score);
		} else  if (score > 0) 
		{
			power += tokens[tail--];
			score--;
		} else 
		{
			break;
		}
	}
	return maxScore;
}
