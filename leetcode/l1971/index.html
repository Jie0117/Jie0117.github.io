1971. Find if Path Exists in Graph

April 21, 2024

—

題目敘述

There is a bi-directional graph with n vertices, where each vertex is labeled from 0 to n - 1 (inclusive). The edges in the graph are represented as a 2D integer array edges, where each edges[i] = [ui, vi] denotes a bi-directional edge between vertex ui and vertex vi. Every vertex pair is connected by at most one edge, and no vertex has an edge to itself.

You want to determine if there is a valid path that exists from vertex source to vertex destination.

Given edges and the integers n, source, and destination, return true if there is a valid path from source to destination, or false otherwise.

想法

一開始先想到用二維陣列搭配DFS搜尋判斷結果，無奈記憶體超出限制，後來使用adjacency list搭配DFS才順利通過。 但是執行速度有一點慢，上網搜尋了Union Set方法來解這題 DFS和Union Find是兩種不同的演算法技巧，它們在解決問題時的想法和實作方式不同。

  1. DFS： DFS 是一種圖遍歷演算法，用於遍歷圖中的所有節點，並根據需要執行特定的操作。 它透過遞歸或堆疊來實現，在遍歷過程中會不斷地深入到圖中的下一層節點，並在到達最深處後再回溯到上一層節點，直到遍歷完所有節點或滿足特定條件為止。 DFS 通常用於搜尋圖中的路徑、判斷圖的連結性、拓樸排序等問題，它是一種全域性的搜尋演算法，可以遍歷圖中的所有節點，並根據需要進行特定的操作。
  2. Union Find： Union Find是一種資料結構，用於管理元素的集合關係。 它支援兩種基本操作：Find和Union。 透過這兩種操作，可以有效率地判斷兩個元素是否屬於同一個集合，以及將兩個集合合併為一個集合。以連通圖的角度來思考，兩頂點之間互相連通可以視為在同一個set，最後要判斷是否連通只需判斷兩點是否在同一set即可。

總結

雖然 DFS 和Union Find都可以用來解決圖的連通性問題，但它們的應用場景和實作方式有所不同。 DFS 是一種全域性的搜尋演算法，適用於需要遍歷圖中所有節點的場景，而Union Find更適用於處理集合的關係，例如判斷兩個節點是否連通、合併兩個集合等。

DFS

class Solution {
public:
    bool dfs(int node, const vector<vector<int>>& graph, int destination, vector<bool>& visited) {
        if (node == destination) {
            return true;
        }
        visited[node] = true;
        for (int neighbor : graph[node]) {
            if (!visited[neighbor]) {
                if (dfs(neighbor, graph, destination, visited)) {
                    return true;
                }
            }
        }
        return false;
    }
    
    bool validPath(int n, vector<vector<int>>& edges, int source, int destination) {
        vector<vector<int>> graph(n);
        for (const auto& edge : edges) {
            int u = edge[0], v = edge[1];
            graph[u].push_back(v);
            graph[v].push_back(u); // Since it's an undirected graph
        }
        
        vector<bool> visited(n, false);
        return dfs(source, graph, destination, visited);
    }
};

Union Set

class UnionFind {
public:
    vector<int> parent;
    
    UnionFind(int n) {
        parent.resize(n);
        for (int i = 0; i < n; ++i) {
            parent[i] = i;
        }
    }
    
    int find(int x) {
        if (parent[x] == x) {
            return x;
        }
        return parent[x] = find(parent[x]);
    }
    
    void unionSets(int x, int y) {
        int rootX = find(x);
        int rootY = find(y);
        if (rootX != rootY) {
            parent[rootX] = rootY;
        }
    }
};

class Solution {
public:
    bool validPath(int n, vector<vector<int>>& edges, int source, int destination) {
        UnionFind uf(n);
        for (const auto& edge : edges) {
            uf.unionSets(edge[0], edge[1]);
        }
        return uf.find(source) == uf.find(destination);
    }
};
