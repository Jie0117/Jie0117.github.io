使用Hugo以及Github建立個人網頁 for Windows

March 6, 2024

—

事前準備

  1. Hugo: https://github.com/gohugoio/hugo/releases (建議下載此版本，for Windows Intel or AMD 64位元CPU https://github.com/gohugoio/hugo/releases/download/v0.123.7/hugo_extended_0.123.7_windows-amd64.zip)
  2. Powershell: https://github.com/PowerShell/PowerShell/releases/tag/v7.4.1 (注意：Windows Powershell跟Powershell是不同的程式，也可以使用WSL)
  3. Git: https://git-scm.com/

Step 1： 建立hugo site

	//建立一個新的hugo專案
	./hugo new site quickstart 

	//移動到quickstart資料夾內
    cd quickstart 			 

	//初始化git
    git init 			
    	 
    //下載ananke這個主題
    git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke 
    
    //在hugo.toml這個檔案告知Hugo我要使用ananke主題
    echo "theme = 'ananke'" >> hugo.toml 
    
    //啟動Hugo的開發伺服器來查看網站
    ./hugo server

Step 2： 新增內容

	//在post這個資料夾建立markdown格式的內容，之後文章就是打在my-first-post.md裡面
	hugo new content posts/my-first-post.md

接下來打開文字編輯軟體你會看到，前面的內容中的draft = true。 預設情況下，Hugo 在建立網站時不會建立草稿內容。

	+++ 
	title = 'My First Post' 
	date = 2024-01-14T07:07:07+01:00 
	draft = true 
	+++

儲存文件，然後啟動Hugo的開發伺服器來查看網站。 可以執行以下任一命令來包含草稿內容。

	./hugo server --buildDrafts 
	./hugo server -D

透過powershell中顯示的 URL(如下列) 查看網站。 當繼續新增和變更內容時，保持開發伺服器運作，會及時顯示在瀏覽器上面。

	Web Server is available at http://localhost:1313/ (bind address 127.0.0.1)

Step 3： 更新配置檔案hugo.toml

打開hugo.toml你會看到下列這些設定，請根據自己喜好更改。舉例來說，baseURL = ‘https://username.github.io/’

	baseURL = 'https://example.org/'
	languageCode = 'en-us'
	title = 'My New Hugo Site'
	theme = 'ananke'

Step 4： 生成網站檔案

請執行下列指令，用來生成網站檔案，生成好的檔案會存在public資料夾裡面

	./hugo

Step 5： 發布到Github上

到此我們已經完成了個人網站了，現在我們要將他發佈到Github的伺服器上面

	// 進入hugo資料夾，進入public資料夾中
	cd public

	// 初始化Git
	git init

	// 將你的網站程式碼添加到這個新的Git
	git add .

	// 提交你的更改，並添加訊息
	git commit -m "Initial commit"

	// 將分支名稱改為 "main"，這是 GitHub 的新預設分支名稱
	git branch -M main

	# 將這個倉庫關聯到你在 GitHub 上創建的倉庫
	git remote add origin https://github.com/username/username.github.io.git

	// 將你的網站程式碼推送到 GitHub
	git push -u origin main

Step 6： 查看網站

接下來到了收穫成果的時候了，到https://username.github.io查看你的個人網站吧！
